## Переменные окружения

Создайте файл `.env` в корне проекта со следующими переменными:

```env
# API Configuration
VITE_APP_API_URL=your_api_url_here
VITE_APP_CENTRY_DSN=your_sentry_dsn_here

# Feature Flags
# Set to 'false' to disable forum functionality
VITE_IS_FORUM_ENABLED=true
```

### Управление функциональностью форума

Для отключения функциональности форума установите переменную окружения:
```
VITE_IS_FORUM_ENABLED=false
```

При отключении форума будут скрыты:
- Вкладка форума в нижней навигации
- Центральная кнопка "+" в нижней навигации
- Кнопка "Новая тема" в модальном окне
- Поиск пользователей на всех страницах (главная, профиль, курсы, уведомления, календарь)
- Секция "Последние темы на форуме" на главной странице
- Секция "Темы пользователя" в профилях (свой и чужой)
- Секция "Друзья" на главной странице и в профилях
- Кнопка "Форум" в меню
- Маршруты форума в приложении
